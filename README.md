# TrueFacts

Bizarre hybrid of an OpenStruct with "black hole" behavior, a write-once disc, and a parody of nerd behavior.

## Installation

Add this line to your application's Gemfile:

    gem 'true_facts'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install true_facts
